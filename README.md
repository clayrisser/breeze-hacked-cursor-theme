## Features

* Compatible with most Linux window managers
* Animated cursors


## Installation

```sh
make install
```


## Dependencies

* [Inkscape](https://inkscape.org)
* [xcursorgen](https://www.x.org/archive/X11R7.7/doc/man/man1/xcursorgen.1.xhtml)
